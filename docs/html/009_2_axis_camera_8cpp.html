static int CameraOpenSocketAndIssueAuthorizedRequest ( const char *  serverName,
const char *  request 
)
static

Opens a socket.

Issues the given http request with the required added information and authentication. It cycles through an array of predetermined encrypted username, password combinations that we expect the users to have at any point in time. If none of the username, password combinations work, it outputs a "Unknown user or password" error. If the request succeeds, it returns the socket number.

Parameters:
serverNameThe information about the host from which this request originates
requestThe request to send to the camera not including boilerplate or authentication. This is usually in the form of "GET <string>"
Returns:
int - failure = ERROR; success = socket number;
