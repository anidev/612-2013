UINT32 EXPORT_FUNC nUsageReporting::report ( tResourceType  resource,
UINT8  instanceNumber,
UINT8  context = 0,
const char *  feature = NULL 
)

Report the usage of a resource of interest.

Parameters:
resourceone of the values in the tResourceType above (max value 51).
instanceNumberan index that identifies the resource instance.
contextan optional additional context number for some cases (such as module number). Set to 0 to omit.
featurea string to be included describing features in use on a specific resource. Setting the same resource more than once allows you to change the feature string.
